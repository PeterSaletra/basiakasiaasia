# Frontend

Ten katalog zawiera aplikację SPA projektu `BasiaKasiaAsia` zbudowaną w React, TypeScript i Vite.

Pełna dokumentacja projektu znajduje się w [../README.md](../README.md).

## Szybki start

```bash
npm install
cp .env.example .env
npm run dev
```

## Najważniejsze komendy

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Konfiguracja

Przed uruchomieniem uzupełnij `frontend/.env` na bazie `frontend/.env.example`.
